# My Planner — Free PWA Setup Guide

A free daily + monthly task planner that works on your phone and laptop like a native app.

---

## Step 1: Put the files on GitHub Pages (free hosting)

1. Go to https://github.com and sign in (or create a free account)
2. Click the **+** button → **New repository**
3. Name it exactly: `planner`
4. Set it to **Public**, then click **Create repository**
5. Click **uploading an existing file** on the next screen
6. Upload ALL these files (keep the folder structure):
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icons/icon-192.png`
   - `icons/icon-512.png`
7. Click **Commit changes**
8. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)**
9. Click **Save** — wait ~1 minute

Your app will be live at:
**https://YOUR-GITHUB-USERNAME.github.io/planner**

---

## Step 2: Install on your phone

### Android (Chrome):
1. Open the link above in Chrome
2. Tap the **Install** banner that appears at the top, OR
3. Tap the 3-dot menu → **Add to Home screen**

### iPhone (Safari):
1. Open the link in Safari
2. Tap the **Share** button (box with arrow)
3. Scroll down → tap **Add to Home Screen**
4. Tap **Add**

---

## Step 3: Install on your laptop

### Chrome / Edge:
1. Open the link in your browser
2. Look for the install icon (⊕) in the address bar, OR
3. Click the 3-dot menu → **Install Planner**

---

## Features
- ✅ Add tasks for any day
- ✅ Strike off completed tasks
- ✅ Flag important tasks (amber border)
- ✅ Push tasks to the next day
- ✅ Month calendar view with dot indicators
- ✅ Works offline
- ✅ Dark mode support
- ✅ Tasks saved locally on your device
